### https://a-sung.github.io/limbus-library/
게임 개발사 프로젝트 문이 제작한 림버스 컴퍼니의 팬메이드 도감 웹 서비스입니다.<br>
이름, 공격 유형, 죄악, 스킬 키워드로 필터링하여 인격 정보를 확인할 수 있습니다.
